# Import examples for RenaceGest

Use these files to test the mass import flow in this order:
1. guardias-orden-1.csv
2. secciones-orden-2.csv
3. pertrechos-orden-3.csv
4. grupos-orden-4.csv
5. mensajes-orden-5.csv

Recommended setup:
- Log in with the hidden administrator or a Maestre account.
- Use the PRUEBA database profile.
- In the import screen, keep delimiter mode on auto or semicolon.
